Twilio WhatsApp
Webhook URL:
{{ request.host_url.rstrip('/') }}/integrations/whatsapp/twilio
Set as "When a Message Comes In" in Twilio Console

Email Inbound {{ 'Configured' if email_configured else 'Not Set Up' }}

Webhook (SendGrid / Mailgun / Postmark)
Inbound Webhook URL:
{{ request.host_url.rstrip('/') }}/integrations/email-inbound
Configure this as your inbound parse webhook in your email provider.
IMAP Polling
Poll URL (call via cron):
{{ request.host_url.rstrip('/') }}/integrations/email-poll?key=YOUR_IMAP_POLL_KEY
Schedule via cron, Task Scheduler, or a service like cron-job.org every 5 minutes.
Set IMAP_HOST, IMAP_USER, IMAP_PASSWORD, and IMAP_POLL_KEY in your .env file.
